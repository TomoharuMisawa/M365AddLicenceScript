テナント提供サービス一覧取得スクリプト、サービス設定スクリプト一式

[前提条件]
PowerShell2.0以上（Windows7以降）
Install-ModuleコマンドレットにてMSOnlineをインストールするため、PSGalleryの信頼が必要
コマンド実行は管理者権限が必要
Microsoft365テナントログインIDには管理者権限が必要

[利用方法]

【概略】
テナント提供サービス一覧取得を行い、テナントで利用可能なライセンスおよび機能をCSVとして一覧取得します。
一覧取得したライセンスおよび機能のうち、不要なライセンス、機能に関する行を削除し、
サービス設定スクリプトにUPNおよび上記一覧を設定することで、必要となるライセンスを付与することができます。

【手順】
１．SetExLicence1_テナント提供サービス一覧取得.ps1起動。
２．MSOnlineモジュールの導入を聞かれた場合、インストールする旨を選択。
３．Microsoft365テナントへのログイン。
４．テナントで利用可能なライセンスおよび機能一覧を保存するファイル名を選択。
５．上記ファイルを確認し、ユーザーへ与える必要の無いライセンスおよび機能の行を削除。
６．SetExLicence1_テナント提供サービス設定.ps1起動。
７．MSOnlineモジュールの導入を聞かれた場合、インストールする旨を選択。
８．Microsoft365テナントへのログイン。
９．上記で保存、編集したライセンスおよび機能一覧を選択。
１０．ライセンスを付与する対象を記したUPN一覧を選択。
１１．日時-set-msoluserlicense.csvがテナント保有のライセンスの数だけできるので適用されたことを確認。
